OpenGL Project

The project consists of the implementation of forward and deferred rendering pipelines. It was developed in the Real-time Graphics module during the Bachelor's degree in Audiovisual Systems Engineering. It was developed using the framework provided by the university and it was a group project composed by 2 people.

Features:
• SSAO+ implementation.
• Irradiance implementation.
• Reflection implementation.
• Skybox implementation.
• Volumetric Lightning implementation.
• Tone mapping implementation.
